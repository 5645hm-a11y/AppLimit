# AppLimit

AppLimit is a focused usage-limit platform that helps block distracting apps, enforce schedules, and keep devices on track. It is built for parents, students, and teams who want clear digital boundaries without complicated setup.

## Downloads

- Windows Installer (ZIP): https://firebasestorage.googleapis.com/v0/b/kiosklimit.firebasestorage.app/o/AppLimit-Desktop-Setup.zip?alt=media
- Android APK: https://firebasestorage.googleapis.com/v0/b/kiosklimit.firebasestorage.app/o/AppLimit.apk?alt=media

## What it does

- Real-time app blocking with clear feedback
- Time schedules, daily limits, and screen-time rules
- Grayscale mode to reduce impulse usage
- Windows desktop installer + Android APK
